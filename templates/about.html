Mon parcours

J'ai débuté dans le web en 2010. J'ai conçu à l'époque un logiciel pour Windows pour s'auto-entraîner sur du vocabulaire anglais. Plusieurs de mes amis le trouvaient utile. Je voulais donc créer un site pour promouvoir ce logiciel et le partager. J'ai utilisé un logiciel wysiwyg, avant de me rendre compte que le code généré n'était, disons... pas très propre. J'ai alors décidé de coder le site moi-même.

Cet entraînement m'a donné envie de continuer dans le monde du web. Actuellement en école d'ingénieur, j'ai intégré, par le biais de l'apprentissage, l'entreprise Kernix Software, spécialisée dans le développement d'applications web multi-canal.

Passionné par le high tech, j'effectue beaucoup de veille technologique, notamment via Twitter. Je découvre chaque jour de nouvelles notions, de nouvelles pratiques concernant principalement le développement front-end et back-end.

Récemment, je suis passé d'un hébergement mutualisé chez OVH à un serveur dédié chez Online.net. Je n'avais aucun besoin de performance, mais cela m'a permis de compléter mon bagage de connaissances en y ajoutant l'installation et l'administration d'un serveur dédié.

A propos du site

Ce tout nouveau site (mai 2013) est hébergé sur mon serveur dédié chez Online.net. Le nom de domaine est également enregistré chez Online.

J'ai procédé à une refonte graphique de ce site durant l'été 2013 avec un tout nouveau design inspiré de la tendance "flat". Au niveau front-end, j'ai utilisé l'excellent framework CSS knacss de Raphaël Goetter qui propose un reset CSS ainsi qu'une très bonne base pour un site responsive. Le design, découpé en bloc, est inspiré de diverses maquettes de "flat design" que l'on peut trouver sur internet.

Du côté serveur, le site repose désormais sur Silex (un mini-framework PHP construit au-dessus des composants Symfony) associé au moteur de template Twig.

Un petit easter egg s'est glissé sur la page d'accueil de ce site. Saurez-vous le retrouver ? :)
