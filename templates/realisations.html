Layout PHP

Mai 2013 Open Source

Classe PHP permettant de gérer le layout html, avec doctype, title, meta, etc.

Layout PHP permet de gérer le layout, la partie fixe de votre page html finale, directement en PHP. En effet, il n'est pas toujours évident de gérer le doctype à la main, les balises meta, l'inclusion des styles et scripts, etc. Grâce à Layout PHP, vous pouvez configurer les balises que vous souhaitez ajouter dans la section head.

De par son utilisation, cette classe peut être intégrée à n'importe quel framework PHP, mais sera naturellement tout aussi adaptée à des structures plus simples. Layout PHP est disponible sur GitHub et via Composer.

Pixel Draw

Avril 2013 Projet scolaire Web sockets

Jeu web et mobile, mélange entre Pictionnary et DrawSomething.

Client web du jeu
Application android du jeu

Réalisé lors d'un projet scolaire par 7 personnes en 5 jours. Composé d'un client web (HTML et JS), d'une application Android, ainsi que d'un serveur PHP, le jeu consiste à réaliser un dessin en pixel art, dans un temps donné, pour faire deviner un mot aux autres joueurs. Plus les joueurs trouvent vite, plus ils gagnent de points. Le dessinateur change à chaque tour. Un chat permet au joueur de donner leurs réponses mais aussi de communiquer entre eux.

Pixel Draw repose principalement sur l'utilisation des web sockets. Côté serveur, j'ai utilisé la bibliothèque Ratchet qui fournit une bonne abstraction des web sockets, du protocole WampServer et des modes RPC et PubSub.

Walks in Orleans

Mai 2011 Projet scolaire

Carte virtuelle permettant de découvrir monuments, lieux historiques ou intéressants d'Orléans.

Ce site en anglais, réalisé dans le cadre d'un projet scolaire, se présente sous la forme d'une grille géographique où chaque case correspond à un lieu. Pour chaque lieu, un diaporama ou une vidéo est proposée, permettant de visualiser l'endroit, ainsi que les lieux à proximité. Des informations supplémentaires sont également disponibles sur l'histoire du lieu.
